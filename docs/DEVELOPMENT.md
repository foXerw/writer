# Novel Writer 开发路线图

> 更新时间: 2026-01-16

## 项目概述

一个参考VSCode风格的专业小说写作应用，支持长篇创作、角色管理、世界观设定等功能。

**技术栈**: Electron + React + TypeScript + Vite + Monaco Editor

## 最新变更

### v1.15 (2026-01-16)

**Bug修复与UI优化**:
- 修复preload暴露API不完整问题（chapter rename/reorder/getById, character, setting）
- 修复CharacterPanel删除功能语法错误
- 修复import路径问题
- 优化侧边栏Tab布局（自定义按钮组替代Ant Design Tabs）
- 修复新建项目对话框深色主题样式（标题、输入框、placeholder）
- 优化首页按钮样式统一深色主题
- 项目创建流程改为两步式（选择模板 -> 下一步 -> 配置项目）

---

### v1.16 (2026-01-16)

**功能修复与深色主题优化**:
- 修复项目名称未传递到Workspace页面的问题
- 修复无效项目文件夹错误弹窗样式（黑底白字）
- 修复Modal.error静态方法警告，改用受控Modal组件
- 优化主页布局，充分利用屏幕空间
- 优化章节列表布局（图标与名称同行显示）
- 统一错误弹窗深色主题样式

**技术改进**:
- 添加window.draggable区域支持
- 统一全局深色主题颜色方案（#000000背景、#ffffff文字）
- 添加dark-modal CSS类统一Modal样式
