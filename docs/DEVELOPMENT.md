# Novel Writer 开发路线图

> 更新时间: 2024-01-14

## 项目概述

一个参考VSCode风格的专业小说写作应用，支持长篇创作、角色管理、世界观设定等功能。

**技术栈**: Electron + React + TypeScript + Vite + Monaco Editor
